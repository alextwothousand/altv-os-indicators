# altv-os-indicators
Vehicle indicators (incl hazard lights)

## Dependencies

```
npm i --save esbuild chalk esbuild-copy-files-plugin
```

## How to use

Create a folder for this resource.

Include this build.js in your global esbuild config.

Build.

And run.

Or use it your way.
